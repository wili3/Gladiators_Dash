Fantasy Skybox 1.2.6
------------------------------------------------------------------

	This package contains 168 hand-painted textures for 28 different designs of skyboxes. It comes with sample scripts and demo scene that helps you quickly test and use the colourful skybox in your scene.

	Features: 

		• Hand-painted textures.
		• 28 different designs of skyboxes.
		• 168 High-quiality textures.
		• 168 Mid-quiality textures.
		• 168 Low-quiality textures.
		• Sunny/night skyboxes.
		• Demo scene and scripts.
		• Light & render settings guide.
		• Desktop and Mobile friendly.

		• Support all build player platforms.
		
	Compatible:

		• Unity 4.7.1 or higher.
		• Unity 5.3.4 or higher.

Release notes
------------------------------------------------------------------

	Version 1.2.6 (Released on May 19, 2016)

		• Update GUI Animator FREE to version 1.0.1.
		• Unity 4.7.1 and higher compatible.
		• Unity 5.3.4 and higher compatible.

	Version 1.2.5 (Released on Mar 29, 2016)
	
		• Add Full Screen toggle button.
		• Add Settings button and details panels.
		• Change texture files now use png instead of tga.
		• Fixed GUID conflict with other packages.
		• Improve first person player controller.
		• Update sample scripts.
		• Update Demo scene.
		• Unity 4.6.9 and higher compatible.
		• Unity 5.3.2 and higher compatible.

	Version 1.2.3 (Released on Aug 14, 2015)

		• Update sample scripts.
		• Update skybox textures.

	Version 1.2.2 (Released on Jun 22, 2015)

		• Update demo scene.
		• Update sample scripts.
		• Add UI in demo scene.
		• Add GUI Animator FREE.
		• Unity 5.0.1 and higher compatible.

	Version 1.2.1 (Released on Jul 29, 2014)

		• Add 14 skyboxes.
		• Update Demo scene.
		• Update FSPlayer script.
		• Change texture compression.

	Version 1.2 (Released on Jun 15, 2014)

		• Change package’s name to Fantasy Skybox.
		• Add 2 skyboxes.
		• Rearranged folders for future update.
		• Change texture format to True color.

	Version 1.0.1 (Released on Jun 4, 2014)

		• Update demo scene.
		• Update “Sunny Player.cs” script file.

	Version 1.0 (Initial version, released on May 30, 2014)
